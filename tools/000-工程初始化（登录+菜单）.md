```
psql -U onepiece -d postgres -h 192.168.0.116 -p 5432 -W
密码: yearlay_1987
```

**数据库名称：** `jxc`

**需创建的数据表：**
1. `sys_user`：用户表，字段包括 id、username、password（md5加密）、real_name、role、status、created_at
2. `sys_menu`：菜单表，字段包括 id、parent_id、name、path、level（1/2/3）、sort、status
3. `sys_role_menu`：角色权限关联表，字段包括 id、role、menu_id

**初始化数据：**

## step 2: 在 server 目录下，基于 Python Flask 框架，生成 jxcServer 工程。

**工程结构：**
```
server/jxcServer/
├── app.py              # 启动入口
├── config.py           # 配置文件（数据库连接等）
├── requirements.txt    # 依赖清单
└── api/
    ├── __init__.py
    └── auth.py         # 登录鉴权接口
```

**需要实现的功能：**
1. `POST /api/login`：登录接口，接收 username 和 password，校验通过后返回 token（token 暂时设计为长期有效，使用 JWT 或 UUID 均可）
2. 统一鉴权中间件：后续所有接口请求头需携带 `Authorization: Bearer <token>`，校验失败返回 401
3. `GET /api/menu`：获取当前登录用户有权限的菜单列表，按三级结构返回

---

## step 3: 在 client 目录下，基于 Vue 3 + Vite，生成 jxcClient 工程。

**技术栈：**
- Vue 3（Composition API）
- Vite
- ant-design-vue
- axios（封装请求，统一处理 token 注入和 401 跳转）
- vue-router 4

**工程结构：**
```
client/jxcClient/
├── src/
│   ├── main.js
│   ├── App.vue
│   ├── router/index.js     # 路由配置（含登录守卫）
│   ├── api/auth.js         # 登录接口调用
│   ├── api/menu.js         # 菜单接口调用
│   ├── utils/request.js    # axios 封装
│   ├── views/
│   │   ├── Login.vue       # 登录页
│   │   └── Home.vue        # 主页（含菜单）
│   └── components/
│       └── SideMenu.vue    # 左侧菜单组件
```

---

## step 4: jxcClient 工程，实现登录页面。

**功能要求：**
1. 页面包含：用户名输入框、密码输入框、登录按钮
2. 点击登录，调用 jxcServer `POST /api/login` 接口
3. 登录成功：将 token 存入 `localStorage`，跳转至主页（菜单页）
4. 登录失败：提示错误信息（用户名或密码错误）
5. 路由守卫：未登录时访问任何页面均跳转至登录页

---

## step 5: jxcClient 工程，集成 ant-design-vue，实现菜单主页。

**功能要求：**
1. 使用 `a-layout`（侧边栏 + 顶栏 + 内容区）布局
2. 左侧菜单：调用 `GET /api/menu` 接口，动态渲染三级菜单（使用 `a-menu` 组件）
3. 顶栏：显示系统名称进销存系统及当前登录用户名，支持退出登录
4. 默认进入菜单：`系统维护 → 登陆密码修改`
5. 退出登录：清除 `localStorage` 中的 token，跳转回登录页
