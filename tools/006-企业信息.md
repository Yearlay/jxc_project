# 企业信息

**功能**：实现二级菜单页面
- 系统维护 → 设置企业信息（菜单 id=53，path=/system/company）

---
# 输入
1）UI 风格参考效果图参考：按照 ant-design-vue 风格 AI 自行设计
2）[企业信息]字段说明：名称、电话、地址
3）企业信息业务说明：保存企业信息，全局只有一条数据；第一次访问为空，保存时自动新增；之后保存为修改。
4）上部分UI效果：`<a-alert>` 提示框（请设置您的企业或门店的基本信息，这些信息会显示在您打印的单据上，请确保信息真实有效。）
5）中间部分：使用 `<a-form>` 垂直布局，label 宽度 160px；
   - 企业/门店名称：`<a-input>` 必填
   - 企业/门店电话：`<a-input>` 必填
   - 企业/门店地址：`<a-input>` 必填
   - 所有字段皆为必填；任何一个为空，保存按钮不可点击（`:disabled="!canSave"`）
6）下部分：保存按钮（`<a-button type="primary">`），保存成功后弹出 `<a-message>` 提示"保存成功"。
7）页面加载时自动调用 GET 接口，回填已有数据。

---
# 实现

## step 1: 在 `pg` 目录下，新建 `08_create_company_table.sql`
- 表名：`sys_company`
- 字段：`id SERIAL PRIMARY KEY`, `name VARCHAR(100) NOT NULL`, `phone VARCHAR(50) NOT NULL`, `address VARCHAR(200) NOT NULL DEFAULT ''`, `updated_at TIMESTAMP NOT NULL DEFAULT NOW()`
- 初始化时**不插入数据**
- 同步更新 `00_reinit_db.sh`，追加执行 `08_create_company_table.sql` 的步骤

## step 2: 后端 `jxcServer` 工程，新建 `api/company.py`
- Blueprint 名：`company_bp`
- `GET /api/company/get`：查询唯一一条企业信息；若无数据则返回 `data: null`
- `POST /api/company/save`：upsert 逻辑；若表中无数据则 INSERT，否则 UPDATE（始终操作 `id=1` 那条）；Body: `{ name, phone, address }`；三个字段均不能为空，否则返回 400
- 在 `app.py` 中注册蓝图：`from api.company import company_bp` + `app.register_blueprint(company_bp, url_prefix='/api')`

## step 3: 前端 `jxcClient` 实现 "企业信息" 页面
- 新建 `src/views/CompanyInfo.vue`
- 在 `src/router/index.js`（或路由配置文件）中，在 `/system` 路径下注册子路由：`{ path: 'company', component: () => import('@/views/CompanyInfo.vue') }`
- 页面逻辑：
  - `onMounted` 时调用 `GET /api/company/get` 回填表单
  - `canSave` computed：`!!form.name?.trim() && !!form.phone?.trim() && !!form.address?.trim()`
  - 保存按钮点击后调用 `POST /api/company/save`，成功后 `message.success('保存成功')`
