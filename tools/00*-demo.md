# 输入
1）UI 风格参考效果图参考：`design/效果图/基础信息设置_片区管理.png`（用户管理与片区管理页面风格接近）
2）用户管理字段说明：在数据表`sys_user`（脚本`pg/02_create_tables.sql`），补充字段：操作仓库（关联biz_warehouse.id）、管辖片区（关联biz_area.id）。
3）用户管理业务说明：可【按用户名称】查询用户。创建新用户、修改、密码重置、删除。
4）上部分UI效果：【请输入用户名称】搜索框、创建新用户。
5）中间用户列表：按上面[用户管理字段说明]展示列表，每行最右侧为操作：点击三点图标，显示下拉菜单：修改、密码重置、删除。
6）创建新用户弹窗 + 修改用户弹框 + 密码重置弹框 + 删除弹框：基于ant-design-vue，AI自行设计。
