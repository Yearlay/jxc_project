# 采购业务员

**功能**：实现二级菜单页面
- 销售管理 → POS销售

---
# 输入
1）UI页面风格参考 `AreaManage.vue`（增删改查列表页），无状态列。
2）[POS销售]字段说明：【POS前台销售】中产生的销售记录。参考[前台销售]字段说明：
3）[POS销售]业务说明：按销售单号查询，按客户名称查询。按仓库过滤查询，按业务员查询，按日期范围查询。

`销售单号`、`仓库`、`业务员`、`操作员`、`终端`、`顾客信息`、`商品列表`、`商品总数`、`总金额`、`折后价`、`付款方式`、`支付金额`、`应找金额`、`状态`、`日期`

# 实现
## step 1: 后端 `jxcServer` 工程，创建【采购业务员】相关 API
参考[POS销售]业务说明，设计[POS销售]相关业务的接口。

## step 2: 前端 `jxcClient` 实现 "采购业务员" 菜单页面
UI页面风格参考 `AreaManage.vue`（增删改查列表页）。
- 上部分：两个文字输入框（按销售单号查询，按客户名称查询）；下拉选择框查询（按仓库过滤查询，按业务员查询，按日期范围查询）
- 下部分：显示已经结算完成的销售单列表，需显示：`销售单号`、`仓库`、`业务员`、`操作员`、`终端`、`顾客信息`、`商品列表`（概述商品清单）、`商品总数`、`总金额`、`折后价`、`付款方式`、`支付金额`、`应找金额`、`日期`
